Lambda简化

```java
Runnable task2 = new Runnable() {
    @Override
    public void run() {
        log.debug("hello");
    }
};

Thread t2 = new Thread(task2, "t2");
t2.start();
```

使用简化 直接去掉  @Override public void run 只需要参数列表，然后方法体用->连接， 然后大括号可要可不要（多行语句加上）

```java
Runnable task2 = () -> log.dubug("hello");

Thread t2 = new Thread(task2, "t2");
t2.start();
```
